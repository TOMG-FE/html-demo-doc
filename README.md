# html-demo-doc
自动执行代码示例的文档

## Getting Started

1. 安装依赖

```shell
npm install html-demo-doc --save-dev
```

2. 在由于构建示例的代码中引用该组件:

```js
require('html-demo-doc');
```

3. 参照 html-demo-doc 下的 webpack.config.js, webpack.dev.js 构建 webpack 工程。

4. 构建自己的代码示例文档。
